# codeway-nodejs
## Case of Codeway

Codes in src folder.
You can manage port .env file.

- defined PORT: 3000 and 5050

- url: localhost:3000

### Google Cloud Platform
  - PubSub 
  - Big Query 
  - Dataflow

### Please use following command to your terminal.

- npm install

(It will download dependencies(package.json) for project.)

start command: 

- npm start
